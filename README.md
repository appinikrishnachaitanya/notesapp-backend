# Project Title

notesapp-backend

## Getting Started

this is notesapp-backend project which consists of apis .

## Tech Stack

* Spring Boot
* Java 8
* Maven
* Postgresql
* Junit
* swagger (for Documentation)

### Softwares

The things you need before installing the software.

* Postman
* pgadmin
* SpringToolSuite / Eclipse/ Visual Studio Code
* Java 8 or above
* Git Bash
